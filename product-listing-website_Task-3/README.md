##  About The Project
TechShop is a modern, responsive product listing website built with HTML, CSS and JavaScript. It features a clean, animated interface for displaying tech products with filtering, search and cart functionality.

##  Features
- **Product Listing**: Grid-based product display with hover animations
- **Filtering System**: Category-based filtering (Mobile, Laptop, Watch, Headphones)
- **Search Functionality**: Real-time product search
- **Product Navigation**: Next/Previous buttons for product browsing
- **Shopping Cart**: Local storage-based cart system
- **Responsive Design**: Mobile-first responsive layout
- **Modern Animations**: Smooth CSS animations and transitions
